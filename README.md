# Sistema de Gestión Escolar

Este repositorio contiene una implementación de referencia de un sistema escolar desarrollado con **Laravel 11** y **Filament 3** en el backend, acompañado de un frontend público en **Vue 3**.

## Estructura
- `backend/`: API REST, panel administrativo y migraciones de base de datos.
- `frontend/`: Aplicación SPA para consultas públicas y portales simplificados.
- `docs/`: Diagramas, manuales y documentación técnica.

Consulta los manuales en `docs/` para conocer la arquitectura, instalación y uso del sistema.
